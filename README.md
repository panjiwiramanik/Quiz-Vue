# Vue Quiz

Tugas Pemrograman Web Dinamis Berbasis Vue JS, Menggunakan :
* Bootstrap Vue
* Vue Router
* VUEX

## Getting Started

Bacalah Intruksi Dibawah Ini Agar Projek Dapat Dijalankan Dengan Lancar.

### Prerequisites

Yang Dibutuhkan

```
NPM
```

### Installing

Pertama
```
git clone https://github.com/panjiwiramanik/Quiz-Vue.git
```

Kedua
```
cd Quiz-Vue
```

Ketiga
```
npm install
```

Keempat
```
npm run serve
```
